# TeaTime
========

Tea app. CRUD / Active Record practice.

Includes local `jQuery` and `jQuery` UI library.

After cloning to local store, run these commands in terminal to setup:
`bundle update`
`rake db:reset`
`shotgun`

Utilizing @DandyDanny's custom sinatra skeleton.
